# 智策 AI 协作平台

## 一、项目介绍

基于 Spring Boot 3+Spring AI 构建 AI 智能体，集成多模型调用与 RAG 知识库，实现对话记忆持久化、结构化输出及工具调用（含联网搜索、PDF 生成）。
采用 ReAct 模式实现自主任务规划，支持流式接口与容器化部署，具备恋爱指导、约会计划生成等功能。
